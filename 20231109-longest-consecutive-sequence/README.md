# Longest Consecutive Sequence

## Challenge name: 

Longest Consecutive Sequence

## Description: 

Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

## Example

**Example 1:**
```
Input: S =  [100, 4, 200, 1, 3, 2] 
Output: [1,2,3,4]
```

**Example 2:**
```
Input: S = [1,2,3,6,8,10,5,7]
Output:  [5,6,7,8]
```
